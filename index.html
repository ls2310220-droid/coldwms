📋 Como conectar ao Google Sheets

1. Abra o Google Sheets e crie uma planilha nova
2. Clique em Extensões → Apps Script
3. Apague o código existente e cole o conteúdo do arquivo ColdWMS_AppScript.js
4. Salve (Ctrl+S) e clique em Executar → setup para criar as abas
5. Clique em Implantar → Nova implantação
6. Tipo: App da Web · Acesso: Qualquer pessoa
7. Copie a URL gerada e cole no campo acima
8. Hospede este HTML em qualquer lugar (GitHub Pages, Netlify, pen drive com link local)
💡 Dica: Depois que a URL estiver configurada, todos os operadores usam o mesmo link HTML. Os dados vão direto para o Google Sheets em tempo real.
